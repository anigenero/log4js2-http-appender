Installation

> npm i -S @log4js2/http-appender

Configuration

Configure the appender using the configure() method.

{% highlight typescript %} import {configure, LogLevel} from '@log4js2/core'; import {HttpAppender} from '@log4js2/http-appender'; configure({ level: LogLevel.INFO layout: '%d [%p] %c %M:%line:%column - %m %ex', appenders: [{ name: 'Http', appender: HttpAppender, config: { url: '/log-endpoint' } }], loggers: [{ tag: 'App', logLevel : LogLevel.INFO }] }); {% endhighlight %} {% include sections/server-configuration.html %}
